# Porcio Frontend

Aplicación frontend para Porcio, un sistema de seguimiento de porciones para nutricionistas y pacientes.

## Tecnologías
- React 19
- Vite
- Tailwind CSS 4
- React Router 7
- Lucide React (Iconos)
- Axios (HTTP Client)

## Estructura de carpetas
- `src/pages`: Páginas principales (Login, Dashboards)
- `src/components`: Componentes reutilizables
- `src/context`: Contextos de React para estado global
- `src/services`: Servicios para llamadas a la API
- `src/hooks`: Hooks personalizados

## Ejecución en desarrollo
```bash
npm install
npm run dev
```

El servidor se ejecutará en el puerto 3000 (o 3001 si el 3000 está ocupado).
